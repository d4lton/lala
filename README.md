# lala
A simple language lexical analyzer, parser and interpreter

The language is currently very simple, it only handles numeric variable declaration, IF/ELSE statements and a couple built-in functions.

Example:

<pre>
<code>
a = 1;
b = 3;
c = 4;
if ((a * 4) + 1 == b || c > 2) {
  a = a + 1;
} else {
  hide();
}
</code>
</pre>
